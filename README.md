# AMBER Trial — Attentional Task Analysis
 
Analysis of attention data collected as part of the **AMBER trial**, a crossover RCT investigating VR-based binocular stimulation as a treatment for residual amblyopia (Simon-Martinez et al., 2023, *BMC Ophthalmology*).
  

## Reference

Simon-Martinez et al. (2023). Stereoptic serious games as a visual rehabilitation tool for individuals with a residual amblyopia (AMBER trial). *BMC Ophthalmology*, 23, 220. https://doi.org/10.1186/s12886-023-02944-y


---

 
## Data

Raw data from the AMBER trial is not included in this repository.

## Setup

Requires Python 3.12.

Install dependencies using pipenv:

```
pipenv install
```

Or using pip:

```
pip install -r requirements.txt
```
